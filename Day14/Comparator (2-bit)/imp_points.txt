✅ Important Points – 2-bit Comparator
Definition
A comparator compares two binary numbers and determines if one is greater than, less than, or equal to the other.

Inputs and Outputs (2-bit)

Inputs: Two 2-bit numbers (A[1:0], B[1:0])

Outputs:

gt: A > B

lt: A < B

eq: A == B

Logic Approach

Compare MSB first; if equal, compare LSB.

For example:

If A[1] > B[1], then A > B

If A[1] < B[1], then A < B

If A[1] == B[1], check A[0] and B[0].

Truth Table
Has 16 possible combinations (4 for A x 4 for B).

Applications

Sorting circuits

Priority encoders

Arithmetic circuits requiring magnitude comparison

Design Styles

Behavioral (using if-else)

Structural (using logic gates like AND, OR, NOT for direct gate-level design)
